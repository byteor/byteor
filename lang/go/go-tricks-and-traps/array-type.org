** Code

#+BEGIN_SRC go
package main

import "fmt"

func main() {
	a := [...]int{1, 2, 3, 4}
	b := [...]int{5, 6}

	fmt.Printf("a: %T\n", a)
	fmt.Printf("b: %T\n", b)
}
#+END_SRC

** Run

#+BEGIN_SRC shell
a: [4]int
b: [2]int
#+END_SRC

** Notes

*%T* tell fmt print out the variable's type name. *[4]int* and *[2]int* are two
different types.
