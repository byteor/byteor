** Code

#+BEGIN_SRC go
package main

func pr(a [3]int) {
	println(&a)
	for _, i := range a {
		println(i)
	}
}

func main() {
	a := [3]int{1, 2, 3}
	println(&a)

	pr(a)
}
#+END_SRC

** Run

#+BEGIN_SRC shell
go run pr.go
0xc420037f60
0xc420037f48
1
2
3
#+END_SRC

** Notes

Array is value type, so passing by value will copy the whole array.
