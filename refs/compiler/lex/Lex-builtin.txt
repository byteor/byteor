Lex Builtin Names
=================

+------------------------+-----------------------------------------------+
|     Lex Builtin        |                    Description                |
+------------------------+-----------------------------------------------+
| lex.yy.c or lexyy.c    | Lex output file name                          |
+------------------------+-----------------------------------------------+
| yylex()                | Lex scanner routine                           |
+------------------------+-----------------------------------------------+
| yytext                 | Current matched string                        |
+------------------------+-----------------------------------------------+
| yyin                   | Lex input file (Default: stdin)               |
+------------------------+-----------------------------------------------+
| yyout                  | Lex output file (Default: stdout)             |
+------------------------+-----------------------------------------------+
| input()                | Lex buffered input routine                    |
+------------------------+-----------------------------------------------+
| ECHO                   | Lex default behaviour (print yytext to yyout) |
+------------------------+-----------------------------------------------+
