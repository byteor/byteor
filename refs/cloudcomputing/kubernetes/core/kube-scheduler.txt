# kube-scheduler

For given pod:

+----------------------------------+
|        Schedulable nodes:        |
|                                  |
| +--------+ +--------+ +--------+ |
| | node 1 | | node 2 | | node 3 | |
| +--------+ +--------+ +--------+ |
+----------------------------------+
                |
+----------------------------------+
   Pred. filters: node 3 doesn't
   have enough resource
+----------------------------------+
                |
                |
+----------------------------------+
|          remaining nodes:        |
|                                  |
| +--------+            +--------+ |
| | node 1 |            | node 2 | |
| +--------+            +--------+ |
+----------------------------------+
                |
+----------------------------------+
  Priority function: node 1: p=2
                     node 2: p=5
+----------------------------------+
                |
 select max{node priority} = node 2
